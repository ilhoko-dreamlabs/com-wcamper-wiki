# 리커버리 SOP(팀 운용)

## 한 줄 요약
리커버리는 ‘힘’이 아니라 ‘절차’다.

## 내용
### SOP 목표
- 실패를 줄이고, 부상을 막고, 차량 손상을 최소화한다.

### 절차(요약)
1) 상황 평가: 왜 못 나가는가(트랙션/차체 걸림/경사/진흙)
2) 인원 통제: 라인/윈치 라인 주변 접근 금지
3) 장비 정격 확인 + 연결(포인트/샤클/스트랩)
4) 각도/앵커 확보(필요 시 스내치 블록)
5) 단계적 시도: 파내기 → 트랙션보드 → 견인/윈치
6) 종료 후 점검: 포인트/장비/차량 손상 확인

### 금지 행동(필수)
- ‘고리 아무 데나’ 연결
- 라인 위/근처에 사람 서기
- 키네틱 로프를 설명/절차 없이 사용

## 체크리스트(실전)
- 역할 배정(리드/지휘, 스팟터, 장비 담당)
- 안전거리 확보/통제
- 단계적 접근(힘보다 원인 제거)

## 태그(예시)
- TeamKit=RECOVERY
- Ops=RecoverySOP
- SafetyRisk=HIGH
