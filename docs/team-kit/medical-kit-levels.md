# 의료/외상 키트 표준(레벨 1~3)

## 한 줄 요약
의료는 장비보다 ‘점검/절차/역할’이 먼저다.

## 내용
### 레벨 표준(권장)
- **Level 1(기본)**: 밴드/거즈/소독/테이프/진통제(개인)
- **Level 2(팀 공용)**: 압박붕대, 삼각건, 보온 담요, 가위, 장갑
- **Level 3(외상/Trauma)**: 출혈 대응(교육/절차 포함), 지혈 도구 등(팀 합의 필요)

### 운영 원칙
- 키트는 ‘보유’보다 **사용 가능**이 중요(훈련/점검).
- 역할: “의료 담당(가능한 사람)” 지정 권장.

## 체크리스트(실전)
- 팀 공용 의료 키트 위치 공유
- 내용물 유통기한/소모품 점검
- 의료 담당/연락 프로토콜 정하기

## 태그(예시)
- TeamKit=MEDICAL
- SafetyRisk=HIGH(사고 시)
