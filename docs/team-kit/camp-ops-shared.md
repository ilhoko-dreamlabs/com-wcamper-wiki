# 공용 캠프 운영(쉘터/조리/쓰레기)

## 한 줄 요약
팀 캠프는 ‘편의’보다 ‘민원/안전/환경’이 먼저다.

## 내용
### 공용 캠프의 목적
- 회의/식사/휴식 공간(공용 쉘터/타프)
- 조리 동선 단순화(공용 키친)
- 민원/환경 리스크 최소화(쓰레기/소음/조명)

### 권장 공용 구성
- 공용 쉘터/타프(강풍 시 철수 기준 포함)
- 공용 조리(버너/테이블/물)
- 쓰레기 분리/보관 시스템(반출 전제)
- 조명은 ‘국소’ 중심(빛공해 최소)

### 운영 규칙
- 매너타임/조명 다운
- 화기/소화수단 표준

## 체크리스트(실전)
- 공용 쉘터 설치/철수 담당 지정
- 화기 안전거리/소화수단 확인
- 쓰레기 반출 계획(봉투/보관함)
