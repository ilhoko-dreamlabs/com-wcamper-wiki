# 팀 공용 장비(컨보이) 개요

오버랜딩은 차량 세팅도 중요하지만, **여러 팀이 함께 이동**할 때는  
공용 장비(Shared Kit)와 프로토콜(SOP)이 성공률을 좌우합니다.

## 이 섹션이 다루는 것
- 통신: **CB/무전** 기본, 프로토콜(콜사인/위험 콜)
- 내비: 오프라인 지도/GPX 표준, 좌표 공유 규격
- 리커버리: 팀 단위 표준 키트 + 금지 행동 포함 SOP
- 전력/비상: 점프/부스터/공용 충전 허브
- 의료/안전: 레벨별 키트 표준 + 역할
- 캠프 운영: 공용 쉘터/조리/쓰레기 시스템
- 운영 체계: 역할(리드/스윕/스팟터), 체크인/로그
