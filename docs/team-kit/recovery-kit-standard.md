# 팀 리커버리 키트 표준(공용)

## 한 줄 요약
리커버리는 장비보다 ‘절차’와 ‘정격/인원 통제’가 먼저다.

## 내용
### 목표
팀 단위로 ‘최소 공용 리커버리 키트’를 표준화하면,
사고/매몰 대응 속도가 크게 올라갑니다.

### 권장 공용 구성(예시)
- **Kinetic rope** 또는 적절한 견인 스트랩(정격)
- **Soft shackle / Bow shackle**(정격)
- **Tree saver** + **Winch extension**
- **Snatch block**(필요 시)
- **Traction boards**(팀 합계 2~4세트 이상)
- **Gloves/보호구**, **Winch damper**
- **Jack base plate**(연약 지면)

### 금지(표준)
- 정격 미확인 장비 사용
- 라인 근처 인원 접근
- 감정/즉흥으로 ‘한 번에 세게’ 당기기

## 체크리스트(실전)
- 팀 공용 장비 담당자 지정(소유/관리/적재 위치)
- 정격 표기 확인(차량 중량 대비 여유)
- ‘금지 행동’ 공유(라인 근처 접근 금지)

## 태그(예시)
- TeamKit=RECOVERY
- SafetyRisk=HIGH
