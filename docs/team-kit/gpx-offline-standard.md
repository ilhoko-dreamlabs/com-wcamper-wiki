# 오프라인 지도/GPX 운영 표준

## 한 줄 요약
오프라인 지도/GPX는 고립 환경에서 ‘길을 잃지 않게’ 하는 핵심 인프라.

## 내용
### 목표
- 통신이 약해도 **경로를 잃지 않게** 운영 표준을 만든다.

### 표준 항목(권장)
- **GPX 트랙**: 날짜/루트명/버전(v1, v2) 포함
- **대체 경로(Exit/Bypass)**: 최소 1개
- **포인트(웨이포인트)**: 합류/재집결/주유/보급/위험지점

### 좌표 공유 규칙(권장)
- WGS84 Decimal(예: 37.123456, 127.123456)로 통일
- “도/분/초” 혼용 금지(혼선 방지)

### 공용 저장소 운영
- 파일명 규칙 + 변경 로그(누가/언제/무엇을)

## 체크리스트(실전)
- 출발 전: 오프라인 지도 다운로드 완료
- 트랙/대체 경로/웨이포인트 확인
- 좌표 표기 규칙 팀 통일(WGS84 Decimal)
