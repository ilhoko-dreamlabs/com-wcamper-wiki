# 표준 키트(5대 기준)

## 운영 원칙(5대 기준)
- 팀이 커지면 ‘장비’보다 **역할/체크인/재집결 규칙**이 더 중요해진다.
- 공용 장비는 **중복(예비)** 개념이 들어가야 한다(단일 고장=전체 리스크).

|---|---:|---:|---|
| 통신 | 차량용 무전/CB | 1 | 채널 통일 |
| 안전 | 소화기(차량) | 1 | |
| 의료 | Level 1 | 1 | |

| 리커버리 | 기본 스트랩 또는 보드 | 1 | 팀 표준화 권장 |

## 운용 표준(5대에서 반드시)
- **재집결 포인트(웨이포인트)**를 최소 3개(아침/점심/야간) 설정
- **간격 규칙**: 앞차 시야 상실 시 “LOST VISUAL” 콜 → 정지/재집결
- **체크인**: 2시간 주기(험로/통신 약하면 1시간)
- **리커버리 지휘자(Recovery Lead)** 지정(한 명이 통제)
