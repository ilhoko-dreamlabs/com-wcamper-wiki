# 무전 프로토콜(콜사인/위험 콜)

## 한 줄 요약
무전은 ‘대화’가 아니라 ‘운영’이다. 위험 콜 우선, 짧고 표준화.

## 내용
### 목표
- 컨보이에서 무전은 ‘대화’가 아니라 **운영**이다.
- 짧고, 표준 용어로, 우선순위가 명확해야 한다.

### 표준 프로토콜(권장)
- **콜사인 먼저 → 메시지 짧게 → 필요 시 반복**
- 위험 상황은 최우선:
  - **STOP**: 즉시 정지(선두/후미 모두)
  - **HAZARD**: 위험 요소(낙석/세굴/보행자 등)
  - **GAP**: 차량 간격이 벌어짐
  - **LOST VISUAL**: 앞차 시야 상실(정지/재집결)

### 메시지 포맷 예시
- “스윕 → 리드, GAP 2대 발생, 다음 교차로에서 홀딩 요청”
- “리드 → 전원, HAZARD 낙석, 속도 10으로 다운”

### 금지
- 장시간 잡담(채널 혼잡)
- 욕설/감정 표현(분쟁 촉발)

## 체크리스트(실전)
- 채널/볼륨/마이크 위치 점검
- 위험 콜 3개(STOP/HAZARD/GAP)만 먼저 숙지
- 합류/이탈 시 ‘누가 빠졌는지’ 콜로 확인
