# 출발 전 브리핑 템플릿

## 목적
출발 전 10분 브리핑으로 **사고/혼선/분쟁**을 줄인다.

## 브리핑 템플릿(복붙용)
- 날짜/루트명:
- 참가 차량 수 / 그룹 분할(있으면):
- 리드/스윕/리커버리/의료/통신 담당:
- 메인 채널 / 예비 채널:
- 오늘 목표 구간(시작→중간→도착):
- 재집결 포인트(웨이포인트):
- 체크인 주기(시간/방식):
- 철수 기준(풍속/강우/수위/피로):
- 고위험 구간(낙석/세굴/도하/모래):
- 리커버리 장비 분산(누가 무엇):
- 캠프 운영(공용 쉘터/취사/쓰레기):

## 운영 팁
- “누가 무엇을 결정하는지(권한)”를 명확히 한다.
- 합류/이탈은 무조건 콜로 확정한다.
