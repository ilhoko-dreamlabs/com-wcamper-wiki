# CB 라디오 기본

## 한 줄 요약
CB는 팀 운영의 기본 통신 수단이며, 규칙과 안테나 세팅이 품질을 좌우합니다.

## 내용
### 핵심 개념
- **CB(Citizens Band)**는 차량 간 **단거리 음성 통신**에 쓰이는 대표 수단입니다.
- 장점: 즉시성(통화 연결 불필요), 팀 단위 운영(채널 공유)
- 한계: 지형/거리/안테나/설치 품질에 따라 품질 편차가 큼

### 표준 용어(위키 연결용)
- **CB Radio** — 시민밴드 무전(차량 간 음성 통신)
- **Channel(채널)** — 팀이 공유하는 통신 채널(출정 전에 고정)
- **Callsign(콜사인)** — 차량/역할 식별 호칭(리드/스윕 등)
- **PTT** — Push-To-Talk(버튼 누르고 말하기)
- **Antenna(안테나)** — 송수신 성능을 좌우(설치/튜닝 중요)
- **SWR** — 안테나 튜닝/매칭 지표(높으면 성능/장비 보호 문제)

### 운영 원칙
- 장비보다 **규칙(프로토콜)**이 먼저다.
- ‘잘 들리냐’보다 **위험 콜(정지/장애물/이탈)**이 우선한다.

## 체크리스트(실전)
- 출정 전: 채널 1개 고정 + 예비 채널 1개
- 콜사인 규칙 합의(리드/스윕/차량번호)
- 위험 콜(STOP/HAZARD) 표준 문구 합의
- 안테나 고정/케이블 상태 확인
