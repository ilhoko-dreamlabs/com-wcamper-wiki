# 표준 키트(2대 기준)

## 운영 원칙(2대 기준)
- **각 차량 개인 장비(필수)** + **팀 공용 장비(최소)**로 구성
- “장비 보유”보다 **담당자/적재 위치/절차(SOP)**가 중요
- 체크인/이탈/리커버리는 무조건 **표준 문구** 사용

## 차량별 필수(각 차량 1세트)
| 카테고리 | 항목 | 최소 | 비고 |
|---|---:|---:|---|
| 통신 | 차량용 무전 또는 CB(기본) | 1 | 채널/콜사인 통일 |
| 전력 | 휴대 충전/케이블 | 1 | 팀 공용 허브와 연동 |
| 안전 | 소화기(차량) | 1 | 접근 쉬운 위치 |
| 의료 | Level 1(개인 기본) | 1 | 밴드/거즈/소독 등 |
| 리커버리 | 리커버리 포인트/연결 확인 | - | “아무 곳 견인” 금지 |
| 타이어 | 디플레이터/게이지 | 1 | 공기압 표준 운영 |
| 조명 | 헤드랜턴 | 1 | 야간 작업 필수 |

## 팀 공용(2대 기준 최소 공용 세트)
| 카테고리 | 항목 | 권장 수량 | 담당(예시) | 비고 |
|---|---|---:|---|---|
| 통신 | 예비 마이크/휴대 무전 1대 | 1 | Sweep | 고장/분실 대비 |
| 내비 | GPX 트랙(본/대체) + 오프라인 지도 | 1 세트 | Lead | 파일명 규칙/버전 |
| 리커버리 | 트리세이버 | 1 | Recovery | |
| 리커버리 | 소프트 샤클 | 2 | Recovery | 정격 표기 확인 |
| 리커버리 | 스내치 스트랩(정격) | 1 | Recovery | 키네틱은 숙련 시 |
| 리커버리 | 윈치 댐퍼 | 1 | Recovery | |
| 리커버리 | 트랙션보드 | 2세트 | Each | 최소 1세트/차 권장 |
| 리커버리 | 삽/스페이드 | 1 | Recovery | 파내기가 먼저 |
| 전력 | 점프스타터(대용량) | 1 | Sweep | 충전상태 점검 |
| 전력 | 부스터 케이블(정격) | 1 | Sweep | |
| 의료 | Level 2(팀 공용) | 1 | Medical | 압박붕대/가위 등 |
| 캠프 | 공용 타프/쉘터(소형) | 1 | Camp | 강풍 시 철수 기준 |
| 환경 | 쓰레기 보관(봉투/박스/집게/장갑) | 1 | Camp | 100% 반출 |

## 최소 프로토콜(2대일수록 중요)
- **채널/콜사인**: “Lead / Sweep” + 차량번호
- **위험 콜**: STOP / HAZARD / GAP 3개만 우선 표준화
- **체크인**: 2~4시간마다 위치/상태 보고(통신 약하면 더 촘촘히)
- **철수 기준(Abort)**: 풍속/강우/수위/피로 중 하나라도 기준 초과 시 즉시 조정
