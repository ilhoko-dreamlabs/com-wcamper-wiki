# 오버랜딩 세팅: 운영 SOP(필드 표준)

## 한 줄 요약
세팅의 완성은 장비가 아니라 ‘운영 표준(SOP)’이다.

## 핵심 용어(표준)
- **Lead / Sweep** — 컨보이 선두/후미 역할
- **Spotter** — 장애물 구간 유도자
- **Check-in schedule** — 정기 안전 보고 주기
- **Comms protocol** — 무전/메신저 표준 호칭·콜
- **Recovery SOP** — 리커버리 표준 절차(정격·각도·안전거리)
- **Abort criteria** — 철수/중단 기준(풍속/수위/피로 등)
- **Pace plan** — 일정 페이스/도착 마감 계획
- **Escape route** — 탈출로/회차 공간 확보
- **Incident log** — 사고/분쟁 기록(사실/정책/감정 분리)
- **Fallback spot** — 대체 스팟 확보(규정/기상 대응)

## 체크리스트(실전)
- 출발 전: 역할(리드/스윕/스팟터), 통신 규칙, 철수 기준 합의
- 운영 중: 체크인(좌표/상태) 주기 유지, 이탈/합류 규칙 적용
- 리커버리: 절차/인원 통제 없으면 ‘시도하지 않는 것’이 원칙

## 태그(예시)
- Ops=SOP
- SafetyRisk=MID~HIGH
