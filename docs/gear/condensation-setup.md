# 결로 대응 셋업

## 한 줄 요약
결로는 ‘난방 부족’이 아니라 ‘환기·습도·온도차’ 문제다.

## 핵심(운영 관점)
- 난방 강할수록 수증기 증가 → 결로 증가 가능.
- **상단+하단 2점 환기**로 대류를 만들어야 함.
- 젖은 장비(옷/신발)는 외부/전실로 분리.

## 체크리스트(실전)
- 상단/하단 환기 포인트 확보
- 젖은 장비 분리 보관
- 아침 건조 루틴(문 열기/햇빛)

## 태그(예시)
- SafetyRisk=MID~HIGH(동계)
- Setup=Ventilation
