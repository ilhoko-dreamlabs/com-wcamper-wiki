# 쉘터 vs 리빙쉘

## 한 줄 요약
거실 중심(쉘터)과 거주 강화(리빙쉘)는 목적이 다르다.

## 핵심(운영 관점)
- 쉘터: 거실/작업 공간 확보(개방성)
- 리빙쉘: 체류/우천/동계에 유리(거주성)
- 운영 포인트: ‘크기’보다 **환기/바람/철수 루틴**이 중요

## 체크리스트(실전)
- 체류 목적(우천/동계/장박) 명확화
- 강풍 시 철수 기준 정하기
- 결로/환기 설계 확인

## 태그(예시)
- Gear=Shelter/LivingShell
- SafetyRisk=MID(강풍)
