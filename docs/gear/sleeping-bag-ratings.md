# 침낭 온도 표기(컴포트/리밋)

## 한 줄 요약
침낭 스펙은 ‘최저기온’이 아니라 컴포트/리밋 기준으로 해석해야 한다.

## 핵심(운영 관점)
- 브랜드 표기 온도는 해석이 다름(표준/비표준 혼재).
- 실무 팁: **‘컴포트 온도’를 목표로 잡고 여유(2~5°C)를 둔다.**
- 내부 습기/결로가 보온력에 직접 타격.

## 체크리스트(실전)
- 침낭 표기에서 ‘Comfort’ 기준 확인
- 추가 레이어(라이너/담요/의복) 계획
- 아침에 건조 루틴(침낭 펼침/환기)

## 태그(예시)
- Gear=SleepingBag
- Seasonality=WINTER(조건부)
