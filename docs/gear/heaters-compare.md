# 난방(가스/등유/전기) 비교

## 한 줄 요약
난방은 따뜻함보다 ‘환기·CO·화재’가 핵심이다.

## 핵심(운영 관점)
- 가스: 간편하지만 환기/결로/CO 리스크
- 등유: 출력 좋지만 연료/냄새/환기/화재 리스크
- 전기: 안전하지만 전력/차단/정책 제약

## 체크리스트(실전)
- 환기/CO 경보기 확보
- 취침 전 난방 OFF 원칙(또는 안전 설계)
- 캠핑장 전열 정책/차단 확인

## 태그(예시)
- SafetyRisk=HIGH(조건부)
- Seasonality=WINTER
