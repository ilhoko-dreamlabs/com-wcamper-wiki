# 타프 셋업(각도/배수/풍하중)

## 한 줄 요약
타프는 예쁘게가 아니라 ‘배수/바람’이 먼저다.

## 핵심(운영 관점)
- 우천 시 **물길(배수)**가 핵심: 처짐/고임 방지.
- 강풍 시 면적을 줄이고 낮추는 것이 안전.
- 팩/가이라인 각도와 텐션 유지가 품질을 결정.

## 체크리스트(실전)
- 비 올 때 물 고임 포인트 제거(경사)
- 바람 방향으로 면적 줄이기
- 라인 텐션 재점검(야간 포함)

## 태그(예시)
- Gear=Tarp
- Weather=RAIN/WIND
