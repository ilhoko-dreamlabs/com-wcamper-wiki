# 수면 시스템(개요)

## 한 줄 요약
침낭/매트/의복/환기의 조합이 ‘따뜻함’과 ‘결로’를 결정한다.

## 핵심(운영 관점)
- 수면 체감은 **온도 + 바람 + 습도(결로)**의 합.
- 침낭 단독이 아니라 **매트(R-value) + 바닥 + 환기**가 체감온도에 더 큰 영향을 줄 때가 많음.
- 동계에서 ‘따뜻하지만 젖는 셋업’은 최악(결로 → 보온력 하락).

## 체크리스트(실전)
- 오늘 최저기온/체감온도 확인
- 매트 R-value가 계절에 적절한지
- 침낭/이불이 젖지 않게 환기/젖은 장비 분리

## 태그(예시)
- Seasonality=ALL
- SafetyRisk=MID(동계)
