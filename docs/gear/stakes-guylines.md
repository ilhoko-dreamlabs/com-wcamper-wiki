# 팩/가이라인(정격/각도)

## 한 줄 요약
고정 실패가 장비 파손·부상으로 이어진다.

## 핵심(운영 관점)
- 팩은 지면(데크/자갈/흙)에 따라 최적이 다름.
- 가이라인은 45도 내외로, 텐션 유지가 중요.
- 강풍 예보 시 ‘고정 강화’보다 ‘면적 축소/철수’가 우선일 때가 많음.

## 체크리스트(실전)
- 지면 타입별 팩 준비
- 가이라인 마모/매듭 확인
- 강풍 시 철수 기준 합의

## 태그(예시)
- Gear=Stakes/Guylines
- SafetyRisk=MID~HIGH
