# 공구/스페어 키트(기본)

## 한 줄 요약
현장 문제의 대부분은 작은 소모품으로 해결된다. 공구보다 스페어의 준비가 복구 시간을 줄인다.

## 핵심(운영 관점)
- 고장 자체보다 "복구 불가"가 일정 전체를 무너뜨린다.
- 차량/캠핑 장비 공통으로 자주 끊기는 소모품(퓨즈, 밸브, 라인, 커넥터)을 우선 준비한다.
- 팀 운영에서는 공구를 분산하지 말고 표준 키트로 묶는 것이 효율적이다.

## 체크리스트(실전)
- 퓨즈/케이블/커넥터/테이프 등 소모품 보유
- 타이어 펑크 대응(플러그/컴프레서/게이지) 준비
- 수리 후 테스트 절차(누설/발열/체결) 확인

## 태그(예시)
- Gear=Tools/Spares
- Ops=RecoveryReadiness
