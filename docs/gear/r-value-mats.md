# 매트 R-value(바닥 단열)

## 한 줄 요약
동계 체감의 절반은 ‘바닥 단열’에서 결정된다.

## 핵심(운영 관점)
- 바닥으로 빠지는 열 손실이 매우 큼.
- R-value는 단열 성능 지표(높을수록 따뜻).
- 실무: 동계는 **단일 고R 매트 또는 2중(폼+에어) 조합**이 안정적.

## 체크리스트(실전)
- 오늘 환경(지면/데크/파쇄석/눈) 고려
- R-value 목표 설정(계절별)
- 에어매트 누기 대비(패치/예비)

## 태그(예시)
- Gear=SleepMat
- SafetyRisk=MID(저체온)
