# 리스크 태그 매트릭스(안전/규정 분리)

## 한 줄 요약
안전 리스크와 규정 리스크는 분리해야 운영이 쉬워진다.

## 왜 중요한가
- **SafetyRisk**: 생명/부상/재산 피해 가능성
- **ComplianceRisk**: 단속/민원/소유권/정책 충돌 가능성

## 체크리스트(실전)
- 동일 장소라도 Safety/Compliance가 다르게 나올 수 있음
- ‘합법/불법’ 단정 대신 Risk 태그로 기록

## 태그(예시)
- SafetyRisk=LOW/MID/HIGH
- ComplianceRisk=LOW/MID/HIGH
