# Ford Ranger

## 한 줄 정의
**ROUTE+UTILITY**(장거리+모듈화) 성향의 리그. 적재·수납·전력 세팅이 유리하다.

## Best fit 미션
- ROUTE ★★★★☆
- TRAIL ★★★☆☆ (트림/타이어/보호 세팅에 따라)
- FAMILY ★★★☆☆ (2열/적재 방식 고려)
- HEAVY ★★★★☆ (픽업 적재/모듈 확장에 강점)

## 핵심 스펙 체크(운영 관점)
- 적재: 픽업 베드 기반 모듈(서랍/키친/탑퍼) 구성 용이
- 견인: 트레일러/카라반 운용 시 **정격/수직하중/스웨이**가 핵심
- 구동: 4WD/로우레인지 유무는 트림/사양에 따라 확인

## 세팅 난이도(현실)
- 수납/모듈: 쉬움(하지만 무게가 늘기 쉬움)
- 수면: 베드 텐트/탑퍼/RTT 등 선택지 다양
- 도시 운용: 길이/회차/주차가 데일리 난이도를 올릴 수 있음

## 추천 로드아웃(예시)
- 라이트: AT 타이어 + 기본 리커버리 + 베드 정리
- 미들: 탑퍼 + 전력 + 냉장고 + 물
- 헤비: 키친 모듈 + 장기 자급(물/연료/전력) → **Payload/CoG** 필수 관리

## 주요 리스크
- 베드 적재 과적 → 서스/제동/타이어 정격 문제
- 견인 운용 시 스웨이/제동 리스크
- 오프로드 구간은 하부 보호/리커버리 포인트 확인 필요

## 태그 요약(초안)
- Chassis=BODY_ON_FRAME
- Drivetrain=4WD(사양별)
- LowRange=사양별 확인
- PayloadClass=MID~HEAVY
- TowingClass=MID~HEAVY
- BuildBias=ROUTE/HEAVY
- EcosystemScore=4~5 / ServiceScore=3~4(지역 의존)
