# Jeep Wrangler Rubicon

## 한 줄 정의
**TRAIL 중심 오버랜딩**에 강한 레퍼런스 리그. 단, **적재/장거리 편의**는 세팅으로 보완해야 한다.

## Best fit 미션
- **TRAIL** ★★★★★
- ROUTE ★★★☆☆ (타협 필요)
- FAMILY ★★☆☆☆ (공간/승차감 고려)
- HEAVY ★★☆☆☆ (페이로드/CoG 관리 중요)

## 핵심 스펙 체크(운영 관점)
- 구동: 4WD + (차종에 따라) **Low range**
- 트랙션: (차종/옵션에 따라) 락/트랙션 지원
- 지오메트리: 접근/이탈/브레이크오버 강점이 목적 적합성을 높임

## 세팅 난이도(현실)
- 전력: 보조배터리/퓨즈블록은 비교적 단순
- 수납: 서랍/랙 넣으면 페이로드 압박 → “필요 최소” 전략 권장
- 수면: RTT는 편하지만 CoG 상승 → 경량/저상 세팅 우선 고려

## 추천 로드아웃(예시)
- 라이트: AT 타이어 + 기본 리커버리 + 경량 캠프
- 미들: 하부 보호 + 윈치(필요 시) + 전력/냉장고
- 헤비: RTT/서랍/물 시스템 → **총중량/무게중심** 숫자 관리 필수

## 주요 리스크
- 과적(페이로드) + 루프 적재(CoG 상승)
- 장거리 피로(정숙/승차감) 관리 필요
- 대형 타이어/리프트는 “성능”보다 “손상/정비/안정성” 트레이드오프를 문서화

## 태그 요약(초안)
- Chassis=BODY_ON_FRAME (차종에 따라)
- Drivetrain=4WD
- LowRange=YES(트림/사양별 확인)
- DiffLock=REAR 또는 FRONT+REAR(트림/사양별 확인)
- BuildBias=TRAIL
- EcosystemScore=5 / ServiceScore=3~4(지역 의존)
