# 오버랜딩 차량 선정 프레임

## 먼저 결정할 것: 미션 프로필
- **ROUTE(루트/장거리)**: 고장 리스크↓, 연료/주행 편의↑, 적재·수면 운영↑
- **TRAIL(험로/기술 주행)**: 로우레인지/락/지오메트리, 하부 보호, 리커버리 중심
- **FAMILY(가족/장기 체류)**: 실내 공간/안전/편의, 전력·물 시스템 확장
- **HEAVY(자급 확장)**: 페이로드/견인/수납 모듈, 전력·물·연료를 숫자로 관리

> 핵심: **모델 나열보다 ‘미션 프로필’이 먼저**입니다.

## 차체 구조: 세팅 확장성의 뼈대
- **BODY-ON-FRAME(프레임 바디)**: 적재/견인/험로 내구성에 유리(대신 승차감/연비 타협)
- **UNIBODY(유니바디)**: 온로드 편의/연비 유리(대신 과적/험로 지속은 설계 따라 편차)

## 구동계 최소 기준
- 최소: **AWD/4WD + 적절한 타이어/지상고**
- 험로 비중↑: **Low range(로우레인지)**, 락(또는 강한 트랙션), 리커버리 포인트, 하부 보호

## 오버랜딩에서 더 중요한 현실 체크
- **페이로드/총중량**: RTT, 서랍, 냉장고, 물/연료 넣으면 금방 한계
- **정비/부품 생태계**: 장거리에서는 성능보다 **복구 가능성**
- **적재/수면 운영성**: “내부 공간/평탄화/수납 동선”이 체감 품질을 좌우

## 추천 태그(위키 표준)
- Chassis: BODY_ON_FRAME / UNIBODY
- Drivetrain: AWD / PART_TIME_4WD / FULL_TIME_4WD
- LowRange: YES/NO
- DiffLock: REAR / FRONT+REAR / NONE
- PayloadClass: LIGHT / MID / HEAVY
- TowingClass: LIGHT / MID / HEAVY
- Size: COMPACT / MID / FULL
- BuildBias: ROUTE / TRAIL / FAMILY / HEAVY
- EcosystemScore: 1~5(부품/커뮤니티/튜닝)
- ServiceScore: 1~5(정비 접근성)

## 페이지 템플릿(차량 페이지 공통)
1) 한 줄 정의(왜 이 차가 오버랜딩에 유리한가)  
2) Best fit 미션(ROUTE/TRAIL/FAMILY/HEAVY)  
3) 핵심 스펙 체크(구동/로우/락/지상고/타이어)  
4) 세팅 난이도(전력/수납/수면/리커버리)  
5) 추천 로드아웃(라이트/미들/헤비)  
6) 리스크(페이로드/CoG/정비/연료/부품)  
7) 태그 요약  
8) 메모(국내 가용성/정비망/커뮤니티)
