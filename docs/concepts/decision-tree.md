# 판정 트리(Decision Tree)

아래 질문 5개로 대부분의 케이스가 분류됩니다.

## Q1. 여정(루트)이 주인공인가?
- Yes → **오버랜딩**(루트/구간/경로가 컨텐츠 본체)
- No → 다음

## Q2. 험로 주행 자체가 목적(기술/장애물)인가?
- Yes → **오프로드**
- No → 다음

## Q3. 지정·관리·예약(운영 주체)이 있는가?
- Yes → 캠핑장/지정 야영장
- No → **노지(비지정)**

## Q4. 접근/통신/구조가 어려운가?
- Yes → **오지(고립도 높음)** 태그 추가

## Q5. 규정 리스크가 높은가?
- Yes → **ComplianceRisk=HIGH**로 분리 표기(합법/불법 단정 금지)
