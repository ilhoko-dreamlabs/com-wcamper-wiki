# 트레일 (Trail)

## 한 줄 정의
험로 주행을 목적으로 정의된 비포장 경로/구간(난이도와 규칙이 중요).

## 동의어/별칭
Trail

## 포함 / 제외
### 포함
- 장애물/경사/노면 변화 존재
- 스팟터/컨보이 운영 가능

### 제외
- 일반 비포장 도로(정비된 자갈길)

## 판정 기준(운영 기준)
- 장애물/험로가 주행의 본체인가?
- 난이도/규칙을 공유하는가?

## 태그(예시)
- DrivingTerrain=TRAIL
- TrailDifficulty=1~5

## 유사/대조 용어
- [임도 (Forest road)](forest-road.md)
- [오프로드 (Off-road)](offroad.md)

## 흔한 오해(반례)
‘임도=트레일’은 오해. 임도는 관리되는 산림도로인 경우가 많다.
