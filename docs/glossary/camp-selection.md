# 캠프 선정 기준

## 한 줄 정의
바람/수위/경사/지면/탈출로를 기준으로 ‘안전한 숙영 지점’을 선택하는 실무 기준.

## 동의어/별칭
캠프 스팟

## 포함 / 제외
### 포함
- 바람 방향/풍하중 고려
- 수위/우천 시 배수
- 탈출로/회차 공간

### 제외
- 뷰만 보고 결정

## 판정 기준(운영 기준)
- 강풍·우천 시 위험 요소가 있는가?
- 차량이 빠져나갈 수 있는가?

## 태그(예시)
- SafetyRisk=MID
- Remoteness=var

## 유사/대조 용어

- [노지 (Non-designated)](nondesignated.md)

## 흔한 오해(반례)
‘사람 없으면 좋은 스팟’은 오해. 고립/구조/수위 리스크가 더 크다.
