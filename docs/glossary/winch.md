# 윈치 (Winch)

## 한 줄 정의
전동/유압 장치로 케이블을 감아 차량을 끌어 당기는 리커버리 장비. 앵커/각도/정격이 중요.

## 동의어/별칭
Winch

## 포함 / 제외
### 포함
- 자력/상호 견인
- 앵커(나무/바위/매트) 사용

### 제외
- 정격 미확인/사람 접근 상태로 사용

## 판정 기준(운영 기준)
- 앵커가 안전한가?
- 케이블 라인에 사람이 없는가?

## 태그(예시)
- SafetyRisk=HIGH
- VehicleRequirement=WINCH_RECOMMENDED(고립/험로)

## 흔한 오해(반례)
‘윈치가 있으면 무조건 빠져나온다’는 오해. 앵커/환경이 없으면 사용 불가.
