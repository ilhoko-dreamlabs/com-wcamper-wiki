# 자급 (Self-sufficiency)

## 한 줄 정의
외부 인프라 없이 일정 기간 생활 가능한 수준(전력·물·연료·식량·비상)을 의미.

## 동의어/별칭
자급, Self-sufficiency

## 포함 / 제외
### 포함
- 전력/물/연료/식량
- 비상/통신

### 제외
- 장비 과시로만 소비

## 판정 기준(운영 기준)
- 계획된 기간 동안 외부 보급 없이 가능한가?
- 비상 계획이 있는가?

## 태그(예시)
- TripStructure=ROUTE_MULTI_DAY
- SafetyRisk=MID

## 흔한 오해(반례)
‘자급=무조건 많이 싣기’는 오해. 필요한 만큼, 안전한 만큼이 핵심이다.
