# 전력 시스템

## 한 줄 정의
보조배터리/인버터/솔라 등을 통해 캠프 전력을 자급하는 구성. 사용량 산정이 핵심.

## 동의어/별칭
보조배터리, 인버터, 솔라

## 포함 / 제외
### 포함
- 부하(소비전력) 산정
- 충전원(주행/태양광)

### 제외
- 용량/배선/퓨즈 무시

## 판정 기준(운영 기준)
- 일일 Wh를 계산했는가?
- 퓨즈/배선/환기가 안전한가?

## 태그(예시)
- SafetyRisk=MID
- Infrastructure=POWER
