# 디프락 (Diff lock)

## 한 줄 정의
좌우 바퀴 회전 차를 제한해 구동력을 확보하는 장치/기능. 특정 조건에서 탈출력을 개선한다.

## 동의어/별칭
Locker, Locking diff

## 포함 / 제외
### 포함
- 교차축/미끄럼 상황에서 유리
- 저속·험로에서 주로 사용

### 제외
- 일반 주행에서 상시 사용

## 판정 기준(운영 기준)
- 한쪽 바퀴가 뜨거나 헛도는가?
- 저속·험로 상황인가?

## 태그(예시)
- VehicleRequirement=LOCKER_BENEFICIAL(상황별)

## 유사/대조 용어

- [리커버리 (Recovery)](recovery.md)

## 흔한 오해(반례)
‘락만 있으면 트랙션보드 불필요’는 오해. 상황별로 보조 수단이 필요하다.
