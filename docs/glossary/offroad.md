# 오프로드 (Off-road)

## 한 줄 정의
험로 주행(노면 난이도) 자체가 목적. 주행 기술·차량 성능·장애물 주파가 핵심이다.

## 동의어/별칭
Off-road, 트레일

## 포함 / 제외
### 포함
- 트레일/험로 주행이 메인
- 난이도/장애물 중심

### 제외
- 루트 여행이 본체인 오버랜딩(오프로드 구간이 일부인 경우)

## 판정 기준(운영 기준)
- 오늘의 목적이 ‘트레일을 타는 것’이었나?
- 성패가 ‘장애물 통과’에 달렸나?

## 태그(예시)
- DrivingTerrain=TRAIL
- TrailDifficulty=1~5
