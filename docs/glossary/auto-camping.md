# 오토캠핑

## 한 줄 정의
차량 접근 가능한 사이트에서 장비를 내려 설치하는 텐트 중심 캠핑.

## 동의어/별칭
오토

## 포함 / 제외
### 포함
- 차량 접근
- 설치/철수 포함

### 제외
- 차량 내부 수면 중심=차박

## 판정 기준(운영 기준)
- 수면이 텐트인가?
- 차량은 접근 수단인가?

## 태그(예시)

- SpotStatus=DESIGNATED(대개)
