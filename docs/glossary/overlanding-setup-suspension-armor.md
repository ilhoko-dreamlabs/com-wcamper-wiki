# 오버랜딩 세팅: 서스펜션/차체 보호

## 한 줄 요약
세팅은 ‘높이’보다 ‘안전한 제어와 보호’가 목적이다.

## 핵심 용어(표준)
- **Lift(리프트)** — 차체/서스 높이 상승(지오메트리·CoG 영향)
- **Shocks/Struts** — 감쇠 장치(하중·승차감·안정성 영향)
- **Spring rate** — 스프링 강성(하중 대응, 과하면 빈차 승차감 악화)
- **Bump stop** — 스트로크 끝에서 충격 흡수
- **Skid plate** — 하부 보호판(엔진/미션/트랜스퍼)
- **Rock slider** — 사이드 보호+잭/리커버리 포인트 역할
- **Armor(아머링)** — 하부/측면/범퍼 보호 구성 총칭
- **Underbody protection** — 하부 간섭/파손을 줄이는 보호 체계
- **Alignment(얼라인)** — 리프트 후 조향/편마모/안정성에 직결
- **Unsprung weight** — 스프링 아래 무게(승차감/응답성에 영향)

## 체크리스트(실전)
- 리프트는 ‘간섭 해소’와 ‘CoG 상승’이 함께 온다(목표를 명확히)
- 하부 보호는 ‘필요한 구간’ 중심으로 단계적 적용(무게 증가 주의)
- 얼라인/편마모/제동 성능을 세팅 항목으로 포함
