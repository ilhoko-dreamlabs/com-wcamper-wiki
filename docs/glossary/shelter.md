# 쉘터

## 한 줄 정의
거실/작업 공간을 만드는 외피 구조물(텐트와 달리 ‘거주 공간’ 중심으로 쓰이기도 함).

## 동의어/별칭
Shelter

## 포함 / 제외
### 포함
- 바람/비 차단
- 거실 공간

### 제외
- 수면 전용 텐트로만 정의

## 판정 기준(운영 기준)
- 주 용도가 거실/공간 확보인가?
- 텐트와 결합해 쓰는가?

## 태그(예시)
- SetupLayer=1~2
- SafetyRisk=LOW
