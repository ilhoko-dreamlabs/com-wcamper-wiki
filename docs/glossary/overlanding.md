# 오버랜딩 (Overlanding)

## 한 줄 정의
**여정(루트) 자체가 목적**인 장거리 육상 여행 방식. 이동 중 숙박(캠핑 포함)을 결합한다.

## 포함 / 제외
- 포함: 멀티데이 루트, 여러 지점 경유, 포장/비포장 혼합
- 제외: 단일 캠핑장 왕복(목적지가 본체), 트레일만 타는 오프로드 체험

## 판정 기준
- '루트/구간/경로'가 컨텐츠의 본체인가?
- 캠핑은 루트의 노드(기착)인가?

## 태그(예시)
- TripStructure: ROUTE_MULTI_DAY
- DrivingTerrain: PAVED/FOREST_ROAD 혼합 가능
