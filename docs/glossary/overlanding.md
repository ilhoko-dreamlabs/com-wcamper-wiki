# 오버랜딩 (Overlanding)

## 한 줄 정의
여정(루트) 자체가 목적인 장거리 육상 여행 방식. 이동 중 숙박(캠핑 포함)을 결합한다.

## 동의어/별칭
Overlanding, 루트 여행

## 포함 / 제외
### 포함
- 멀티데이 루트/구간 이동
- 여러 지점 경유
- 포장/비포장 혼합

### 제외
- 단일 캠핑장 왕복
- 험로 주행만 하는 당일 트레일

## 판정 기준(운영 기준)
- 루트/구간/경로가 컨텐츠의 본체인가?
- 캠핑은 루트의 노드(기착)인가?

## 태그(예시)
- TripStructure=ROUTE_MULTI_DAY
- DrivingTerrain=Mixed

‘비포장 많이 타야 오버랜딩’은 오해. 포장 비중이 높아도 루트 중심이면 성립한다.
