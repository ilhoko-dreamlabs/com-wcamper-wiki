# 오버랜딩 세팅: 무게/하중/지오메트리

## 한 줄 요약
세팅 안전의 핵심은 ‘정격(하중)과 무게중심’이다.

## 핵심 용어(표준)
- **GVWR/GVM(총중량)** — 허용 총중량(차량+사람+짐)
- **Payload(페이로드)** — 총중량-공차중량=적재 여력
- **GCWR/GCM(총결합중량)** — 차량+트레일러 결합 허용 총중량
- **Tongue weight(수직하중)** — 트레일러가 히치에 누르는 하중
- **CoG(무게중심)** — 루프 적재/리프트가 안정성에 주는 핵심 변수
- **Approach/Departure angle** — 접근/이탈각(범퍼 간섭 이전 각도)
- **Breakover angle** — 중앙 걸림(휠베이스 영향) 관련 각도
- **Ground clearance** — 최저지상고(하부 간섭 위험)
- **Dynamic vs Static roof load** — 주행 중/정차 시 루프 정격이 다를 수 있음
- **Overweight(과적)** — 정격 초과 상태(제동/타이어/서스 위험)

## 체크리스트(실전)
- 로드아웃 확정 후 ‘총 적재량’(대략이라도)을 계산
- 루프 적재는 CoG 상승/풍하중 증가 → 최소화/고정 강화
- ‘서스 업그레이드’는 정격을 바꾸지 않는다는 점을 명확히 기록

## 태그(예시)

- SafetyRisk=MID
