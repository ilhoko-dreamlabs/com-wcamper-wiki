# 데크 사이트

## 한 줄 정의
목재/합성 데크 바닥의 사이트. 팩 고정 방식이 달라 셋업 영향이 크다.

## 동의어/별칭
Deck

## 포함 / 제외
### 포함
- 데크 전용 팩/고정 필요
- 수평/배수 장점

### 제외
- 일반 흙/파쇄석 팩 방식

## 판정 기준(운영 기준)
- 데크 고정이 가능한가?
- 바람에 대한 고정이 충분한가?

## 태그(예시)
- Ground=DECK
- SafetyRisk=MID(강풍)
