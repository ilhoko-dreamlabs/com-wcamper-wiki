# 타프

## 한 줄 정의
비/햇빛을 차단하는 천막(차양) 구조. 셋업 난이도는 바람/팩킹에 크게 좌우된다.

## 동의어/별칭
Tarp

## 포함 / 제외
### 포함
- 차양/비막
- 다양한 폴/라인 셋업

### 제외
- 수면 공간을 반드시 제공하는 구조

## 판정 기준(운영 기준)
- 바람 방향/고정이 가능한가?
- 우천 시 배수/처짐 관리가 되는가?

## 태그(예시)
- SetupLayer=1
- SafetyRisk=LOW~MID(강풍)
