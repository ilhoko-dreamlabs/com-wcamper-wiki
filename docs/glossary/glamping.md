# 글램핑

## 한 줄 정의
설치/철수 없이 시설이 준비된 숙박형 캠핑 상품(침구/가구/편의 제공).

## 동의어/별칭
Glamping

## 포함 / 제외
### 포함
- 사전 설치
- 편의 제공

### 제외
- 사용자 설치 기반 오토캠핑

## 판정 기준(운영 기준)
- 운영자가 시설/침구를 제공하는가?
- 사용자 설치 부담이 거의 없는가?

## 태그(예시)
- SpotStatus=DESIGNATED
- Infrastructure=HIGH
